# News-Keywords

## 👩‍💻 프로젝트 소개
1. News API (south korea api)를 활용한 일별 뉴스 키워드를 생성합니다.
2. 주제별(경제, 정치, 사회, 문화 등)로 키워드 언급 횟수에 따라 시각화해줍니다.
3. 해당 키워드를 선택하면 관련 뉴스 기사의 타이틀을 보여주고 타이틀 선택시 URL을 통해 해당 기사 전문을 확인 할 수 있도록 streamlit을 통해 구현하는 것이 목표입니다.
